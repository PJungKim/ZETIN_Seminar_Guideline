# ZETIN Seminar Guideline 2023

- 2023년에 진행한 세미나를 바탕으로 세미나 진행을 정리한 문서입니다.
- 개인정보 보호를 위해 PPT 자료는 업로드하지 않습니다. PPT 자료는 ZETIN 홈페이지를 참고 바랍니다.
- 세미나 리스트 내용은 순서가 바뀌어도 상관 없습니다.
